# Strong Standard — Personal Dashboard

Dashboard personal de hábitos y competencia entre Bernardo y Manu.

- **Tracking diario:** calorías, verduras, comidas, pasos, peso y gym.
- **Competencia:** sistema de puntos 100% basado en hábitos. El veredicto con consecuencias es **semanal (domingo)**.
- **Vistas:** marcador head-to-head, día a día, historial de semanas, calendario mensual y progreso de peso.

## Uso

Abre `index.html` en el navegador (o entra a la versión publicada en GitHub Pages). Los datos se guardan localmente en el navegador; usa el botón **Respaldo** para exportar/importar.

## Stack

HTML + CSS + JavaScript (sin dependencias externas). Pensado para correr como página estática en GitHub Pages.
